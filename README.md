# 📚 StudyHub - Smart Semester Organizer

A powerful web app to organize your studies by semester, track subjects, upload documents, add summarized notes, and generate comprehensive semester summaries.

## ✨ Features

- **Multi-semester support** - Create and manage multiple semesters
- **Subject management** - Add subjects with completion criteria (Exam/Project/Both)
- **File upload** - Upload PDFs, PPTs, documents, and images
- **Summarized notes** - Add notes for each file and overall subject notes
- **Smart summaries** - Generate semester-wide summaries combining all notes
- **Data persistence** - All data saves automatically in your browser
- **Export functionality** - Export summaries as text files

## 🚀 How to Use

1. **Create a semester** - Click "New Semester" and give it a name
2. **Add subjects** - Enter subject names and select completion type
3. **Select a subject** - Click on any subject to manage its content
4. **Upload files** - Add your study materials (PDFs, PPTs, etc.)
5. **Add notes** - Write summaries for each file and overall subject notes
6. **Generate summary** - Click "Generate Full Summary" to see everything combined

## 💾 Data Storage

All data is stored locally in your browser using localStorage and IndexedDB. Your files and notes persist even after closing the browser.

## 🛠️ Deployment to GitHub

1. Create a new repository on GitHub
2. Upload all files (index.html, style.css, script.js, README.md)
3. Go to Settings → Pages
4. Select "main" branch as source
5. Your site will be live at `https://[username].github.io/[repository-name]`

## 📱 Mobile Friendly

Works on all devices with responsive design.

## 🔒 Privacy

All data stays on your device. No servers, no tracking, no accounts needed.
